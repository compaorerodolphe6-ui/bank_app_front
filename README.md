Application web front-end pour la gestion bancaire.

## 📄 Pages
- `index.html` — Accueil
- `retrait.html` — formulaire de retrait sur compte
- `depot.html` — formulaire de dépôt sur compte
- `history.html` — Historique de transaction



## 🌿 Branches
- `main` → branche principale
- `feature/ci_cd_added` → branche dédiée à l'ajout de ci.yaml
- `feature-cfa ` → une branche pour la modification de la devise par défaut 
- `feature-historique` → une branche pour l'ajout de la page de l'historique de transactions
- `feature-index` → une branche pour l'ajout de la page d'accueil



## ⚙️ CI/CD
- Validation HTML et CSS automatique
- Déploiement automatique sur GitHub Pages
